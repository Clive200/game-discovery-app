# Game Finder

Browse and filter video games using the RAWG API.

### Deployed link

`https://game-finder-q3rs77llb-takudzwas-projects-605efbbe.vercel.app/`

### What I built

- A React + TypeScript + Vite app with a responsive layout.
- Game browsing UI with cards, loading skeletons, and error handling.
- Filters and UI controls (genre list, platform selector, sort selector, and search input).
- Data fetching hooks to load games/genres/platforms from the RAWG API.

### What I learned

This project helped me learn **React**, especially component composition, hooks, props/state, and working with APIs.
